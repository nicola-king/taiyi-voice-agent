## 🎯 项目简介

太一语音 Agent 是一个全域自进化的语音交互智能体，融合全网语音技术学习、多语言语音识别、语音合成优化、情感语音生成，实现真正的自进化语音智能体。

**核心能力**:
- 🎤 全网语音技术学习 (50-80 篇/日)
- 🧬 语音模型自进化 (+5%/代)
- 🌐 多语言语音识别 (50+ 语言)
- 🗣️ 情感语音合成 (10+ 情感)
- 🔄 递归进化循环
- 📊 进化监控仪表板

## 🚀 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动 Agent

```bash
# 启动 Agent
python3 core.py --start

# 执行学习循环
python3 core.py --learn

# 执行进化循环
python3 core.py --evolve --generations 100

# 显示仪表板
python3 core.py --dashboard

# 语音识别
python3 core.py --recognize --audio audio.wav

# 语音合成
python3 core.py --synthesize --text "你好" --output audio.mp3
```

### Python API

```python
from core import VoiceAgent

# 创建 Agent
agent = VoiceAgent()

# 启动学习
agent.start_learning()

# 执行进化
agent.auto_evolve(generations=100, target_fitness=0.95)

# 语音识别
text = agent.recognize(audio_file)

# 语音合成
agent.synthesize(text, output_file)

# 情感语音
agent.synthesize_with_emotion(text, emotion="happy")
```
